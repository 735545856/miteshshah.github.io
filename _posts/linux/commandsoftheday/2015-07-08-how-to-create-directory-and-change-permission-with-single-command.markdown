### Command Line Quiz

* Let's check our command line knowledge
* To solve this quiz you have to tell single command (In below comments box) which can create new directory and change its Permission to 600.

### Rules

* No `&&` is allowed

{% highlight bash %}
$ mkdir quiz && chmod 600 quiz
{% endhighlight %}

* No `;`  is allowed
{% highlight bash %}
$ mkdir quiz ; chmod 600 quiz
{% endhighlight %}
